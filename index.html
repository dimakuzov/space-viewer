Scene Editor

View Mode:
WASD - Move
Q/E - Up/Down
Mouse - Look
Click to lock cursor

Edit Mode:
Click to place panels
Click on panels to edit text
Click on objects to select/delete
Hotkeys:
1 - Add Panel
X/Del - Delete
V - Toggle GLB visibility
C - Toggle Collider Edit
S - Save Transform
Tab - Toggle Mode

Panel Edit Mode:
WASD - Move Panel Horizontally
R/F - Move Panel Up/Down
Ctrl+Enter - Save Changes
Escape - Cancel Editing

Collider Edit Mode:
WASD - Move Collider
Q/E - Move Collider Up/Down
R/F - Scale (±0.5%)
T/Y - Rotate (±0.5°)
5cm per movement
